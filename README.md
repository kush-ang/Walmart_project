# Walmart Sales Analysis Using SQL

## About Project

This project focuses on cleaning and analyzing a Walmart sales dataset using SQL. The objective was to transform raw sales data into a clean and analysis-ready dataset and generate meaningful business insights.

---

## About Walmart

Walmart is one of the world's largest retail companies, serving millions of customers worldwide. Due to its large volume of transactions, Walmart sales data provides valuable insights into revenue trends, customer behavior, profitability, and sales performance.
