Walmart Sales Analysis Using SQL

Project Overview:

This project focuses on cleaning and analyzing a Walmart sales dataset using SQL. The objective was to transform raw sales data into an analysis-ready dataset and generate meaningful business insights through SQL queries.

Project Goal:

The goal of this project was to:

Clean a raw Walmart sales dataset
Improve data quality and consistency
Perform business-focused analysis using SQL
Generate actionable insights related to revenue, profitability, customer behavior, and sales trends

Data Cleaning Process

The following data-cleaning steps were performed:

1. Null Value Check
Checked all columns for missing values
2. Duplicate Record Check
Identified duplicate records
Removed duplicate entries
3. Data Type Validation
Corrected date and time data types
Verified numeric columns
4. Data Consistency Check
Reviewed text values for consistency
Ensured clean and analysis-ready data
